# StudentsxCEOs Website

Official website for StudentsxCEOs - Indonesia's premier leadership accelerator connecting exceptional students with top business leaders.

## Website Structure

- **Home**: `index.html` - Main landing page
- **About**: `about.html` - Organization information and vision
- **Community**: `community.html` - Chapter information and locations  
- **Support**: `support.html` - Partnership and sponsorship opportunities
- **Contact**: `contact.html` - FAQs and contact information

## Deployment

This site is deployed on GitHub Pages.

### Local Development

To run locally, simply open `index.html` in your browser or use a local server:

```bash
# Using Python 3
python3 -m http.server 8000

# Using Node.js
npx http-server
```

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- Font Awesome Icons
